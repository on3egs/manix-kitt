02 — Projet principal

KITT — Copilote vocal IA embarqué

Emmanuel Gelinne est le créateur et concepteur principal d'un système de copilote vocal inspiré de KITT, fonctionnant intégralement en local sur matériel embarqué NVIDIA Jetson, sans aucune dépendance à un service cloud.

Le système reproduit le comportement d'un assistant embarqué intelligent : il comprend la parole en temps réel, raisonne avec un modèle de langage local, répond avec une voix synthétique à effet robotique, et interagit avec l'environnement électronique du véhicule.

Fonctionnalités clés

Reconnaissance vocale Transcription locale en ~300ms via Whisper — aucun envoi vers un serveur cloud
LLM local Qwen 2.5 Raisonnement embarqué en ~1 400ms — entièrement offline, données privées
Synthèse vocale GPU Voix française naturelle en ~490ms via Piper — 5 langues supportées
Personnalité KITT Comportement adapté selon l'utilisateur — loyal, analytique, avec humour
Vision par caméra Détection d'objets en temps réel — KITT voit et décrit son environnement
Réponses instantanées Heure, météo, timer traités en 0ms — sans passer par le LLM
Interface web PWA Installable sur mobile, scanner animé KITT, thème sombre optimisé
Mémoire persistante Retient jusqu'à 50 faits entre les sessions — extraction automatique en conversation
Voix robot emblématique Pitch, overdrive et reverb via Sox DSP — rendu sonore fidèle à la série
Monitor temps réel Surveillance des conversations actives et des stats de connexion via WebSocket
Accès distant sécurisé Tunnel Cloudflare HTTPS — KITT accessible depuis n'importe où dans le monde
Dashboard ZA Elettronica Scanner frontal, voicebox et switchpods authentiques Knight 2000

Architecture

Le système repose sur un pipeline entièrement local : microphone → STT (Whisper GPU ~300ms) → LLM (Qwen 2.5 ~1400ms) → TTS (Piper GPU ~490ms) → haut-parleurs. La latence totale est d'environ 2,3 secondes sur Jetson Orin Nano Super 8GB. Aucune donnée ne quitte le matériel. L'ensemble du code a été conçu et développé par Emmanuel Gelinne.

04 — Stack technique

Technologies utilisées

NVIDIA Jetson Orin Nano Super 8GB — JetPack 6
LLM — Qwen 2.5 3B llama.cpp — inférence GPU locale
STT — Whisper faster-whisper, CUDA float16
TTS — Piper onnxruntime CUDA, 5 langues
Vision — YOLOX-S Apache 2.0 — détection objets temps réel
Audio DSP — Sox Effet voix robot — pitch, overdrive
Backend — Python aiohttp, asyncio, WebSocket
Frontend — PWA Web App Progressive, mobile-first
ZA Elettronica Dashboard Knight Rider professionnel
IoT embarqué GPIO, capteurs, modules distribués
Monitor temps réel WebSocket — conversations en direct
Accès distant Tunnel Cloudflare — accès sécurisé worldwide
05 — Objectifs

Vision et objectifs

L'objectif d'Emmanuel Gelinne est de créer la réplique la plus réaliste possible d'un assistant embarqué de type KITT — autonome, multilingue, fonctionnant localement sans aucune dépendance cloud, sur du matériel accessible.

Au-delà de l'aspect fans, ce projet illustre ce que l'IA embarquée moderne permet : un assistant vocal intelligent, capable d'interaction naturelle, de compréhension contextuelle et d'actions physiques sur le véhicule, le tout dans une architecture compacte et souveraine.

"Créer un KITT réel — pas une simulation, pas un service cloud — une intelligence embarquée dans la voiture, présente, réactive, et fidèle au personnage."

— Emmanuel Gelinne, fondateur KITT Franco-Belge
