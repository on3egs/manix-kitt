# Site web — Emmanuel Gelinne (Manix)

## Ouvrir localement

Double-cliquer sur `ouvrir_site.bat` (Windows)
ou ouvrir `index.html` directement dans le navigateur.

## Publier gratuitement sur GitHub Pages

1. Créer un compte sur https://github.com (si pas encore fait)

2. Créer un nouveau dépôt public :
   - Nom : `manix-kitt` (ou ce que tu veux)
   - Cocher "Public"

3. Uploader les fichiers :
   - Cliquer sur "uploading an existing file"
   - Glisser `index.html` dans la fenêtre
   - Cliquer "Commit changes"

4. Activer GitHub Pages :
   - Aller dans Settings → Pages
   - Source : "Deploy from a branch"
   - Branch : main / (root)
   - Cliquer Save

5. Ton site sera accessible à :
   https://TON_PSEUDO_GITHUB.github.io/manix-kitt/

## Modifier le site

- Ouvrir `index.html` dans n'importe quel éditeur de texte
- Modifier le contenu entre les balises HTML
- **IMPORTANT** : Mettre à jour la section "Historique des modifications" à chaque changement
  (chercher `id="historique"` dans le fichier)
- Re-uploader sur GitHub pour mettre à jour le site en ligne
